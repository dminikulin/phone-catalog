# Phone catalog

This is a React project that represents typical gadget store. It has a responsive and modern design and fast routing.
In this project, you can only view the products. All data is static. However, you can use this project to implement checkout or backend integration by yourself.
The design is based on [this reference](https://www.figma.com/file/7JTa0q8n3dTSAyMNaA0u8o/Phone-catalog-(V2)-Rounded-Style-3?node-id=0%3A1).

### Functionality
1. Pages per each category (Phones, Tablets, Accessories).
2. Favorites page.
3. Cart page.

### Tech stack
**Design:**
- HTML
- CSS (Scss)
- Sass

**Logic:**
- React (with React Router)
- TypeScript (ES6)

### Getting started

If you want to run the project locally, do the following steps:

1. Clone the repository:
```
git clone https://github.com/dminikulin/phone-catalog.git
```
2. Install dependencies:
```
npm install
# or
yarn install
```
3. Run the project in local environment:
```
npm install
# or
yarn install
```
